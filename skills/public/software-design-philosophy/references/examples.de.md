# Anwendungsbeispiele

## Beispiel 1: Chatsystem
User:
Wir bauen ein internes Chatsystem. Bitte ein strukturiertes Design-Outline.

Erwartet:
- Abschnitts-Outline
- Schichten, Module, Zuverlässigkeit

User:
Wandle Schichtung, Modularität und Interface-Verträge in umsetzbare Schritte um.

Erwartet:
- Schritte für Schichten/Module/Verträge

User:
Gib eine Architektur-Review-Checkliste für das Chatsystem (Aktion + Hinweis).

Erwartet:
- Checkliste mit Aktion + Hinweis
