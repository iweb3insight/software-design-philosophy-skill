# Usage Examples

## Example 1: Chat System
User:

Expected:
- Sectioned outline
- Layers, modules, reliability highlights

User:
Turn layering, modularity, and interface contracts into actionable steps.

Expected:
- Actionable steps for layers/modules/contracts

User:
Give me an architecture review checklist for a chat system. Each item must include an action and a tip.

Expected:
- Checklist format with action + tip
