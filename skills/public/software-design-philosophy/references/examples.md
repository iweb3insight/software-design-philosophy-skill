# 使用示例

## 示例 1：聊天系统
用户输入：
我们要做一个企业内部聊天系统，先给一个结构化设计大纲，用于评审。

期望输出：
- 章节化大纲
- 分层、模块、可靠性要点

用户输入：
把分层、模块化、接口契约相关的方法论整理成可执行步骤，用于指导团队开发。

期望输出：
- 分层/模块化/接口契约的行动步骤
- 明确职责与依赖方向

用户输入：
给我一份聊天系统架构评审清单，每条包含行动项和提示。

期望输出：
- 行动项 + 提示的清单格式
