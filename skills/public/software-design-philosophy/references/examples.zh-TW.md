# 使用示例

## 示例 1：聊天系統
使用者：
我們要做一個企業內部聊天系統，請提供結構化設計大綱。

預期：
- 章節化大綱
- 分層、模組、可靠性重點

使用者：
把分層、模組化、介面契約整理成可執行步驟。

預期：
- 分層/模組化/契約的行動步驟

使用者：
給我聊天系統的架構評審清單，每條包含行動與提示。

預期：
- 行動 + 提示清單格式
